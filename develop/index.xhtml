Contributing

XCSoar is a free software project, run by many volunteers in their free time. We need help:

  • Development: can you speak C++? Get the source code, hack it, and submit your patches.
  • Testing: download one of my development builds, test it, and report bugs.
  • Documentation: we need an editor for the XCSoar manual. It is outdated, and needs some care.
  • Translation: check our Launchpad page if you want to help.
