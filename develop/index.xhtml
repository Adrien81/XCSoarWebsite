Contributing

XCSoar is a free software project, run by many volunteers in their free time.
We need help:

  • Development: Can you speak C++? Get the source code, hack it, and submit your patches.
  • Testing: Download XCSoar, test it, and report bugs.
  • Documentation: We need editors for the XCSoar manual. It is outdated, and needs some care.
  • Translation: Check our Launchpad page if you want to help.
