XCSoar Versions:

There are currently three versions of XCSoar that can be used:

  • XCSoar 6.0 (stable)
    Not officially released yet, but in our eyes the best XCSoar that currently exists.
  • XCSoar 6.0 (latest)
    A more up-to-date developer version that might have bugs.
  • XCSoar 5.2.4
    The "old" version which has some known bugs.

XCSoar 6.0 on Windows:

The Windows version of XCSoar can be used to get to know the application or to replay flights with our IGC replay feature.

  • XCSoar 6.0 on Windows (latest)

XCSoar 6.0 on Windows Mobile:

The Windows Mobile version should be used on all PDAs and PNAs. Depending on how old your PDA/PNA is and what operating system is used to run it you need to choose between Pocket PC 2000, Pocket PC 2003 and Windows Mobile 5.

  • XCSoar 6.0 on Windows Mobile 5 (latest)
  • XCSoar 6.0 on Pocket PC 2003 (latest)
  • XCSoar 6.0 on Pocket PC 2000 (latest)

XCSoar 6.0 on Altair:

We are currently not working on XCSoar 6.0 for the Altair platform. If you would like to use XCSoar 6.0 on this platform please contact the manufacturer.

XCSoar 6.0 on Unix:

The Unix version of XCSoar 6.0 is almost ready. We will release a first test version within the next weeks.

XCSoar 6.0 on Android:

We are working on porting XCSoar to the Android platform, but since such a port needs time, please be patient.

Attention

This is just the download of the application itself. In order to use the full potential of XCSoar, you will need Maps for the region you are going to fly in.

Since our Download Wizard is not completed yet, you will have to choose and download the maps by hand. You may use our Terrain Generator to create maps of your area.

An explanation of the files, XCSoar is able to load, is given at the bottom of this page. All the details can also be found in our wiki.

Files

These files can be used with XCSoar

  • Map File
    A map file (.xcm) is a file, that can contain terrain, topology, waypoint and airspace files itself. These files can be created with our Terrain Generator.
  • Airspace File
    An airspace file (.txt) is a file, that contains special use airspace information and coordinates in the OpenAir file format.
  • Waypoint File
    A waypoint file (.xcw / .dat) is a file, that contains waypoints, airports and outlanding fields in the WinPilot/Cambridge waypoint format.
    (Support for SeeYou and Zander files is included in v6.0)
  • Airfields File
    An airfields file (.txt) is a file, that contains detailed information about the loaded waypoints, that is available on a seperate page of the Waypoint Details dialog.
  • Terrain File
    A terrain file (.dat) is a file, that contains all the height information for the chosen area.
    (Outdated! Please use map files instead)
  • Topology Files
    Topology files are files, that contain names and coordinates of cities, roads, railroads, rivers and lakes.
    (Outdated! Please use map files instead)
