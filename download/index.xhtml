Attention

This is just the download of the application itself. In order to use the full potential of XCSoar, you will need Maps for the region you are going to fly in.

Since our Download Wizard is not completed yet, you will have to choose and download the maps by hand. You may use our Terrain Generator to create maps of your area.

An explanation of the files, XCSoar is able to load, is given at the bottom of this page. All the details can also be found in our wiki.

Files

These files can be used with XCSoar

  • Map File
    A map file (.xcm) is a file, that can contain terrain, topology, waypoint and airspace files itself. These files can be created with our Terrain Generator.
  • Airspace File
    An airspace file (.txt) is a file, that contains special use airspace information and coordinates in the OpenAir file format.
  • Waypoint File
    A waypoint file (.xcw / .dat) is a file, that contains waypoints, airports and outlanding fields in the WinPilot/Cambridge waypoint format.
    (Support for SeeYou and Zander files is included in v6.0)
  • Airfields File
    An airfields file (.txt) is a file, that contains detailed information about the loaded waypoints, that is available on a seperate page of the Waypoint Details dialog.
  • Terrain File
    A terrain file (.dat) is a file, that contains all the height information for the chosen area.
    (Outdated! Please use map files instead)
  • Topology Files
    Topology files are files, that contain names and coordinates of cities, roads, railroads, rivers and lakes.
    (Outdated! Please use map files instead)
